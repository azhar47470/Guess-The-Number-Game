# Guess The Number Game

A fun number guessing game built with Python.

## Features
- Generates a random number between 1 and 50
- Player has 7 attempts to guess correctly
- Gives hints — Too High or Too Low after each wrong guess
- Shows number of attempts taken on correct guess
- Reveals answer if player runs out of attempts

## How to Run
1. Make sure Python is installed
2. Run: python Guess_Number_Game.py

## Tech Used
Python — random module, while loops, conditionals
